# Technology Stack

This application is a modern Single Page Application (SPA) built for performance and developer experience.

## Core Technologies
- **Framework**: [React 18](https://react.dev/) - A JavaScript library for building user interfaces.
- **Language**: [TypeScript](https://www.typescriptlang.org/) - Adds static typing to JavaScript for better code quality and tooling.
- **Build Tool**: [Vite](https://vitejs.dev/) - Next Generation Frontend Tooling, ensuring fast development servers and optimized builds.

## Styling & capabilities
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) - A utility-first CSS framework for rapid UI development.
- **Icons**: [Lucide React](https://lucide.dev/) - Beautiful & consistent icons.

## Project Structure
- **Src Directory**: Contains the application logic (`src/`), including components (`App.tsx`, `LandingPage.tsx`) and global styles (`index.css`).
- **Configuration**:
    - `vite.config.ts`: Vite configuration.
    - `tailwind.config.js`: Tailwind CSS configuration.
    - `tsconfig.json`: TypeScript configuration.
